Structure of Marvelmind software package.

Release date 2018.01.04

 

File/folder

Description

dashboard.exe

Dashboard software V5.62.

Portable executable.

install_dashboard_v5_62.exe

Dashboard software V5.62.

Installer file. The installer installs dashboard on the computer, installs serial port driver, places shortcut on the desktop.
